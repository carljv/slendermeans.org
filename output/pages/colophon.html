Code

This site was made with Pelican, a Python-based static site generator. Pelican renders posts and pages written in Restructured Text or Markdown into HTML. The site’s theme was designed using the Jinja templating language.

Code and content for the site were composed in GNU Emacs and Sublime Text 2.

Typography

The body text for the site is Iowan Old Style, designed by John Downer.

Page titles and headings are in Verb Extra Condensed, designed by Ryan Martinson for Yellow Design Studio.

Monospaced text (for example, in code listings) is in Adobe’s Source Code Pro.

Navigation links in the header are in Adobe’s Source Sans Pro. Adobe offers both Source Sans and Source Code for free.

Icons are in the Font Awesome icon font.

Code syntax highlighting is made possible by the epic Pygments project.

Certain other typographic features on the page are facilitated by Christian Metts’s typogrify package.
